# ECommerce Skill für Veronika

## Überblick

Dieser Skill hilft Veronika bei allen ECommerce-Aufgaben – von der Produktrecherche bis zum Kundenservice.

## Funktionen

### 1. Produktrecherche & Lieferantenfindung
- Suche nach Produkten und Lieferanten
- Preisvergleiche
- Bewertungsanalysen
- Dropshipping-Optionen prüfen

### 2. Shop-Management
- Shopify: Produkte hinzufügen, bearbeiten, kategorisieren
- WooCommerce: Produktmanagement
- Lagerbestände verwalten
- Bestellungen tracken

### 3. Marketing & SEO
- Produktbeschreibungen optimieren
- SEO-Keywords recherchieren
- Marketing-Texte erstellen
- Social Media Content

### 4. Kundenservice
- Antwortvorlagen für häufige Fragen
- Retouren-Verwaltung
- Beschwerdemanagement
- FAQ erstellen

### 5. Analytics
- Verkaufszahlen analysieren
- Reports erstellen
- Trends identifizieren

## Schnelle Befehle

```bash
# Produkt recherchieren
node ./skillboss/scripts/api-hub.js search --query "wireless headphones suppliers"

# Bild generieren für Produkt
node ./skillboss/scripts/api-hub.js image --prompt "Professional product photo of wireless headphones on white background"

# Marketing-Text erstellen
node ./skillboss/scripts/api-hub.js chat --prompt "Write compelling product description for wireless headphones"
```
